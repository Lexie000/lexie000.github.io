Projects

Predicting Used Car Prices with Regression Models (R)

MDA9159 Course Project · Western University · Dec 2025

Tools: R, tidyverse (dplyr, ggplot2, tidyr), janitor, caret, broom, car, performance, glmnet, MASS, boot, readr, R Shiny

  • Produced an analysis-ready dataset through structured preprocessing and feature engineering, including quality checks, outlier handling, and standardized categorical variables to support reliable modeling.
  • Developed and compared interpretable multiple linear regression models for log(price), using AIC/BIC and VIF/GVIF to guide model selection and specification testing.
  • Validated robustness via diagnostic checks and cross-validation; delivered a concise report with reproducible outputs, achieving ~75.8% variance explained.

Tags: Regression, Feature Engineering, Model Diagnostics, R

Time Series Forecasting of Cocoa Prices (R)

STA457 Course Project · University of Toronto · May 2025

Tools: R, tidyverse (dplyr, tidyr, ggplot2), forecast, fable, rugarch, tseries, urca, lmtest, zoo, lubridate, readr, rmarkdown

  • Developed forecasting models (ARIMA, SARIMA, ETS, ARIMAX, GARCH) using 30+ years of cocoa futures data.
  • Evaluated models with rolling/walk-forward validation and RMSE/MAE/MAPE; documented assumptions and diagnostics (stationarity, residual checks).
  • Quantified and communicated forecast uncertainty to support risk-aware decisions in commodity planning.
  • Built a reproducible workflow in R using tidyverse, forecast, rugarch, and tseries, with reporting via RMarkdown
GitHub

Tags:Time Series, ARIMA/SARIMA/ETS, GARCH, Forecasting, R
